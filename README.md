# SkyRamp

READ DESC!!

THIS SCRIPT/SOFTWARE IS CREATED AND OWNED FULLY BY ME, RYAN, URMADTEACHER.

This is a "Sky Ramp" that I have scripted, similar to the ones that you see on GTA V.
This code is in LuaU and is meant for Roblox games.
When launching this script, it spawns a sky ramp on you, slightly underneath.
You get different options such as spawning the ramp, deleting or calling an elevator.
The ramp is also customizable, from different colors to different materials.
This script is not meant to be used as an exploit/getting used advantage of
in other games in general. I built this for fun and is meant to be executed
in Roblox Studio OR your own game/s.
